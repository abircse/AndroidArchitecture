# Android Architecture
An Implementation of Google Recommended New Android Architecture with Kotlin

# Uses Technology/Kits
* MVVM
* Coroutines
* Stateflow
* Dagger Hilt
* ViewBinding
* DataBinding
* Lifecycle
* UIState
